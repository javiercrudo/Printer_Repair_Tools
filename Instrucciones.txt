═══════════════════════════════════════════════════════════

PASO 1: DESCARGAR Y DESCOMPRIMIR
---------------------------------
1. Descarga el archivo PrinterRepairTool.zip
2. Haz clic derecho → Extraer todo
3. Elige una carpeta (ejemplo: C:\PrinterRepair\)


PASO 2: PRIMERA EJECUCIÓN
--------------------------
1. Abre la carpeta donde descomprimiste
2. Haz doble clic en: PrinterRepairTool.exe

3. IMPORTANTE: Aparecerá una ventana azul que dice:
   "Windows protegió tu PC"
   
   Esto es NORMAL. Windows muestra esta advertencia
   para todas las aplicaciones nuevas que no están
   firmadas digitalmente.

4. Haz clic en "Más información"
5. Haz clic en "Ejecutar de todas formas"
6. Acepta los permisos de administrador (UAC)


PASO 3: ¡LISTO!
----------------
La aplicación se abrirá normalmente.
Windows recordará tu elección para futuras ejecuciones.


═══════════════════════════════════════════════════════════
  ¿POR QUÉ APARECE ESTA ADVERTENCIA?
═══════════════════════════════════════════════════════════

Windows SmartScreen protege tu PC mostrando advertencias

═══════════════════════════════════════════════════════════
  SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════

PROBLEMA: Windows Defender bloquea el archivo
SOLUCIÓN:
1. Abre Windows Security
2. Ve a: Protección antivirus y contra amenazas
3. Haz clic en "Administrar configuración"
4. Desplázate a "Exclusiones"
5. Agrega PrinterRepairTool.exe como exclusión


PROBLEMA: No aparece el botón "Ejecutar de todas formas"
SOLUCIÓN:
1. Haz clic derecho en PrinterRepairTool.exe
2. Selecciona "Propiedades"
3. En la pestaña "General", marca "Desbloquear"
4. Haz clic en "Aplicar" y "Aceptar"
5. Intenta ejecutar de nuevo
